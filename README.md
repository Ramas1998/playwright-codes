## ⭐ Features

- 🔄 **Auto-detect flaky tests** during Playwright runs  
- 📊 **Dashboard UI** with charts  
- 🔥 **Failure Heatmap** (GitHub-style test × run matrix)  
- 🧠 **AI-style analysis** of failure patterns  
- ⏱ **Run duration timeline**  
- 📉 **Failure trends over time**  
- 📁 **JSON or SQLite storage**  
- 🌗 **Dark mode UI**  
- 💥 **VSCode links** to jump directly to failing tests  
- 🧪 **Supports Playwright retries & multiple projects**
